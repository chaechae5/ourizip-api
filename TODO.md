## pin 기능 구현
임장노트의 핵심 기능. 지도 위에 핀을 찍고 현장 메모를 남깁니다.
### 구현 기능
- [x] 핀 추가 : 클라이언트에서 핀 생성 요청을 보내면 DB에 저장
- [x] 코멘트 작성 : 핀별 텍스트 메모 자유 입력 (수정은 미구현)
- [x] 카테고리 태그 : 카테고리 태그 저장 가능. 교통·학군·상권·소음·주차 등 선택 (수정은 미구현)
- [x] 사진 첨부 : 핀에 사진 1장 이상 첨부 가능 — S3 URL 저장만 구현, Presigned URL 발급 미구현
- [ ] 핀 편집 / 삭제 : 기존 핀 내용 수정 및 삭제 가능
- [ ] 코멘트 목록 : 핀별 코멘트 시간순으로 정렬
- [ ] 핀 검색 : 키워드로 내 핀 검색

### **비기능 요구사항**
- [ ] **오프라인 임시 저장** — 네트워크 없을 때 로컬 저장 후 복귀 시 동기화
- [ ] **이미지 압축** — 업로드 전 1MB 이하로 자동 리사이징
